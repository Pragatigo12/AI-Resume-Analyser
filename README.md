# AI-Resume-Analyser
An intelligent web-based application that analyzes resumes using Artificial Intelligence to provide actionable insights, improve ATS compatibility, and enhance candidate shortlisting efficiency.Resume parsing (PDF/DOCX support)
🤖 AI-based skill extraction and keyword matching
📊 ATS score prediction and feedback
🎯 Job description vs resume comparison
🧠 Smart suggestions for improving resume quality
🔎 Section-wise analysis (Education, Experience, Skills, etc.)
